# SAAB module acronym lexicon (from Tech2Win NAO.bin)

`[A-Z]{2,5} \([A-Z][a-z][^)]+\)`.
**Status:** canonical SAAB-side naming. Does NOT include CAN-ID linkage —
see [`docs/saab_ecu_address_map_2026-05-13.md`](../docs/saab_ecu_address_map_2026-05-13.md)
for the wire-confirmed address mapping built from bench captures.
